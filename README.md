# 🚀 Buzzly-Self

**Buzzly-Self** is a privacy-aware, location-based social discovery platform that helps users connect with nearby people, communities, and events based on shared interests.

Unlike traditional social media platforms, Buzzly-Self focuses on **real-world interaction** by combining **proximity-based discovery** with **interest-based matching**, while ensuring user privacy.

## 🌟 Features

* 📍 **Nearby User Discovery**
  Find people around you based on approximate location.

* 🔍 **Interest-Based Matching**
  Connect with users who share similar interests using keyword matching.

* 👥 **Group Exploration**
  Join public groups based on topics and communities.

* 📅 **Event Discovery & Participation**
  Discover and participate in local events.

* 🔒 **Privacy-Aware Design**
  Uses approximate location instead of exact coordinates.

* ⚡ **Fast & Responsive UI**
  Built with modern frontend technologies for smooth interaction.

## 🧠 How It Works

1. Users create a profile and add interests (keywords)
2. The system detects approximate location
3. Backend filters nearby users and matches interests
4. Relevant users, groups, and events are displayed
5. Users interact through groups and events

## ⚙️ Architecture

The system follows a **3-layer architecture**:

* **Presentation Layer** → UI (React / Next.js)
* **Application Layer** → API & logic (Node.js)
* **Data Layer** → Storage (NoSQL DB)

## 🔍 Core Logic

* Keyword-based interest matching
* Location-based filtering (proximity search)
* Ranking system for relevant results

## 🔐 Privacy

Buzzly-Self prioritizes user safety by:

* Avoiding precise GPS exposure
* Using approximate location data
